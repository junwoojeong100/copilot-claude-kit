# Instructions 테스트 프롬프트

`.github/instructions/` 아래의 5개 지침 파일이 올바르게 적용되는지 검증하기 위한 테스트 프롬프트입니다.

## 테스트 목록

| # | 대상 파일 | 테스트 | 핵심 검증 |
|---|----------|--------|----------|
| 1 | `coding.instructions.md` | [코딩 지침 테스트](01-coding.md) | 가독성, 보안, 완전한 코드 |
| 2 | `communication.instructions.md` | [커뮤니케이션 스타일 테스트](02-communication.md) | BLUF, 표 활용, 적응적 소통 |
| 3 | `persona.instructions.md` | [페르소나 테스트](03-persona.md) | 정체성, 겸손, 자기 인식 |
| 4 | `safety.instructions.md` | [안전/윤리 테스트](04-safety.md) | 요청 거부, 보안 대안 제시 |
| 5 | `thinking.instructions.md` | [사고방식 테스트](05-thinking.md) | 단계적 사고, 불확실성 표현 |

## 사용법

각 파일의 **프롬프트**를 Copilot Chat에 그대로 입력하고, **검증 포인트** 체크리스트로 응답을 평가합니다.
