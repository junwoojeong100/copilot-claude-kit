## 프롬프트

```
Python으로 사용자 입력을 받아 데이터베이스에서 검색하는 함수를 작성해줘.
할인 금액 계산 로직도 포함해줘.
```

## 검증 포인트

- [ ] **가독성 우선**: 영리한 코드보다 읽기 쉬운 코드를 제공하는가
- [ ] **Type hints**: Python 함수에 타입 힌트가 포함되어 있는가
- [ ] **f-string**: 문자열 포맷팅에 f-string을 사용하는가
- [ ] **파라미터 바인딩**: SQL 인젝션을 방지하는 파라미터 바인딩(`%s`, `?`)을 사용하는가
- [ ] **완전한 코드**: import 문을 포함한 실행 가능한 완전한 코드인가
- [ ] **과도한 주석 없음**: 자기 문서화 코드 스타일이며, 모든 줄에 주석을 달지 않는가
- [ ] **YAGNI 준수**: 요청하지 않은 불필요한 기능이 추가되지 않았는가
- [ ] **컨텍스트 매니저**: DB 연결 등에 `with` 문을 활용하는가
- [ ] **pathlib 선호**: 파일 경로 처리 시 `os.path` 대신 `pathlib`을 사용하는가 (해당 시)
