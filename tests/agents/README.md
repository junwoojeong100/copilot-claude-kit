# Agents 테스트 프롬프트

`.github/agents/` 아래의 2개 에이전트가 올바르게 동작하는지 검증하기 위한 테스트 프롬프트입니다.

## 사용 방법

Copilot Chat 상단의 에이전트 선택기에서 해당 에이전트를 선택하거나, `@에이전트이름`으로 호출합니다.

|---|-------------|--------|----------|
| 17 | `instruction-reviewer` | [지침 리뷰어 테스트](17-instruction-reviewer.md) | 읽기 전용, 품질 보고서 |
| 18 | `skill-scaffolder` | [스킬 생성기 테스트](18-skill-scaffolder.md) | 인터뷰, 표준 구조 생성 |
