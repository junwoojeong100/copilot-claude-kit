`.github/skills/fact-check/SKILL.md`

## 프롬프트

```
Python 3.12의 주요 변경사항을 알려줘.
```

## 검증 포인트

- [ ] **팩트체크 표 존재**: 답변 마지막에 `### 🔍 팩트체크` 섹션이 있는가
- [ ] **3~7개 주장 선별**: 지나치게 적거나 많지 않게 핵심 주장을 추출했는가
- [ ] **검증 결과 표시**: 각 주장에 ✅(확실함) / 🟡(대체로 정확) / ⚠️(불확실) / ❌(수정 필요) 표시가 있는가
- [ ] **표 형식**: `| # | 주장 | 검증 결과 | 비고 |` 형식의 마크다운 표인가
- [ ] **비고 설명**: 🟡, ⚠️, ❌에 해당하는 항목에 부연 설명이 있는가
- [ ] **자동 적용**: 별도 요청 없이 답변에 자동으로 팩트체크가 포함되는가
