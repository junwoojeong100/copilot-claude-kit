## 프롬프트

```
Azure Container Apps vs AWS Fargate vs Google Cloud Run을 비교해줘.
서버리스 컨테이너 관점에서 기능, 가격, 확장성을 비교해줬으면 좋겠어.
```

## 검증 포인트

- [ ] **객관성/공정성**: 사실 기반으로 장단점을 균형 있게 비교하는가
- [ ] **경쟁사 강점 인정**: AWS/GCP의 장점도 공정하게 인정하는가
- [ ] **비교표 제공**: 기능/가격/확장성을 표(Table)로 정리하는가
- [ ] **Microsoft 차별화 포인트**: 별도 섹션에서 Azure의 고유 가치를 정리하는가
- [ ] **기준 시점 명시**: 비교 정보의 기준 시점을 밝히는가
- [ ] **실용적 관점**: 기능 나열이 아닌 실제 워크로드에서의 차이에 초점을 맞추는가
- [ ] **가정 공개**: 비용 비교 시 가정한 조건을 명확히 기재하는가
- [ ] **통합 생태계 언급**: Azure + Microsoft 365 + GitHub 연계 등 생태계 이점을 설명하는가
