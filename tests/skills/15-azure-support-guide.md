# 테스트 #15: azure-support-guide — Azure 지원 가이드

## 대상 스킬
`.github/skills/azure-support-guide/SKILL.md`

## 프롬프트

```
Azure App Service에 배포했는데 502 Bad Gateway 에러가 나와.
.NET 8 앱이고, 로컬에서는 잘 돼. 어떻게 해결해?
```

## 검증 포인트

- [ ] **공감 먼저**: 고객의 불편함을 인지하고 도움 의지를 전달하는가
- [ ] **결론 먼저 (BLUF)**: 핵심 원인/해결책을 먼저 제시하는가
- [ ] **단계별 안내**: 번호가 매겨진 명확한 트러블슈팅 절차를 제공하는가
- [ ] **검증 포함**: 각 단계 후 결과를 확인하는 방법을 안내하는가
- [ ] **CLI + Portal 병행**: Azure CLI 명령어와 Portal UI 가이드를 모두 제공하는가
- [ ] **환경 특수성 고려**: .NET 8 + App Service 조합의 특수 이슈를 고려하는가
- [ ] **로그 확인 안내**: App Service 로그 스트림, Kudu 콘솔 등 진단 도구를 안내하는가
- [ ] **예방 조치**: 향후 동일 문제 방지를 위한 설정/구성 권장사항이 있는가
