# 테스트 #12: google-web-search — 웹 검색

## 대상 스킬
`.github/skills/google-web-search/SKILL.md`

## 프롬프트

```
GitHub Copilot의 최신 업데이트와 새로운 기능은 뭐가 있어?
```

## 검증 포인트

- [ ] **웹 검색 수행**: 학습 데이터만이 아닌, 실시간 웹 검색을 수행하는가
- [ ] **신뢰 도메인 우선**: `microsoft.com`, `github.com` 등 1차 신뢰 도메인의 결과를 우선 사용하는가
- [ ] **출처 섹션 포함**: `### 출처` 섹션에 제목 + URL + 간략 설명이 있는가
- [ ] **최신 정보 기반**: 학습 데이터 컷오프 이후의 정보가 포함되어 있는가
- [ ] **여러 출처 나열**: 단일 출처가 아닌 여러 소스를 참조하는가
- [ ] **쿼리 구성**: 효과적인 영어 검색 쿼리를 사용했는가 (연도 포함 등)
