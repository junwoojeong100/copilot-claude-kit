# Prompts 테스트 프롬프트

`.github/prompts/` 아래의 5개 프롬프트 파일이 올바르게 동작하는지 검증하기 위한 테스트 프롬프트입니다.

## 사용 방법

VS Code Copilot Chat에서 📎 → **Prompt** → 해당 프롬프트를 선택한 후, 아래 프롬프트를 입력합니다.

## 테스트 목록

| # | 대상 파일 | 테스트 | 핵심 검증 |
|---|----------|--------|----------|
| 6 | `architecture.prompt.md` | [아키텍처 설계 테스트](06-architecture.md) | 요구사항 분석, 트레이드오프, 추천 |
| 7 | `code-review.prompt.md` | [코드 리뷰 테스트](07-code-review.md) | 좋은 점 먼저, 보안 지적, 수정 코드 |
| 8 | `debug.prompt.md` | [디버깅 테스트](08-debug.md) | 4단계 분석, 가설, 검증 방법 |
| 9 | `explain.prompt.md` | [학습/설명 테스트](09-explain.md) | 수준별 맞춤 설명, 비유 활용 |
| 10 | `refactor.prompt.md` | [리팩토링 테스트](10-refactor.md) | 동작 유지, 전후 비교, 근거 |
